## How to get started?

Connect with your GitHub Copilot by running the command.

```shell
opencode auth login
```

![2026-01-15-at-21.49.332x.png](https://images.nesin.io/f_auto,q_auto/qblog/AIEngineerGuide/2026-01/m1wu1p43n4tzix5llkum)

![2026-01-15-at-21.50.332x.png](https://images.nesin.io/f_auto,q_auto/qblog/AIEngineerGuide/2026-01/caxznoykuw4yifdsyjxc)

Then authenticate it.

Once you're done, you'll be able to choose the model using the `/model` command
