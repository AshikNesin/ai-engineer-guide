## How to get started?

You can install a specific skill say `copywriting` or all the skills together.

To know about all the available skills run the following command

```shell
npx add-skill coreyhaines31/marketingskills --list
```

# Install all skills
npx add-skill coreyhaines31/marketingskills

# Install specific skills
npx add-skill coreyhaines31/marketingskills --skill copywriting
