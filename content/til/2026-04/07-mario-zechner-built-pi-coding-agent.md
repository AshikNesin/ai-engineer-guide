One of my favourite coding agent recently is [Pi](https://pi.dev) - it's a very minimal and highly extensible coding agent.

It barely has any 4 system tools and a very basic system prompt.

And his philosophy in building the Pi is

> Strip away everything, build a minimal, extensible core

```
npm install -g @mariozechner/pi-coding-agent
```
