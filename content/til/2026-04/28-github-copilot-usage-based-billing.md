GitHub Copilot used to request based for AI usage. But now they're moving to Usage Based Billing instead (starting June 1, 2026)

This is an expected move. At the end of the day, if they don't charge based on tokens they're loosing money. And they can subsidize for sometime as a part of their user acquisition strategy.

This happened with Cursor, Trae and pretty much all the AI based IDEs earlier.
