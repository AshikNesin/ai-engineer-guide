Vercel team has released [open-agents](https://github.com/vercel-labs/open-agents) - an open source cloud agent.

Think of like Cursor background Agents which runs on the cloud, does the work for you remote.

{{< video "https://cdn.qblog.nesin.io/qblog/AIEngineerGuide/2026-04/f4qilcyajlvj9opespv1.mp4" >}}

👉 https://open-agents.dev

As you might have guessed, it heavily uses Vercel stack like AI SDK, AI Gateway, Sandbox, Workflow DevKit, etc but since it is a open source, it might be a good reference point if you're building something like this.
