Armin has shared Pi extension that their team at Earendil (which now owns [Pi](https://pi.dev) coding agent) uses for code review.

👉 https://github.com/earendil-works/pi-review

## How to get stared?

```shell
pi install git:github.com/earendil-works/pi-review
```

Once you install it, you'll have `/review` and `/end-review` which you can use to review your changes.

One interesting thing about this code review extension is that unlike other review SKILLs/tools, you get a granular level option for that task that you want to do.
