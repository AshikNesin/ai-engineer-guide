## How to get started?
If you're on Claude Code, you can install it like this

```shell
/plugin marketplace add addyosmani/agent-skills
/plugin install agent-skills@addy-agent-skills
```

For other agents you can refer this [doc](https://github.com/addyosmani/agent-skills/blob/main/docs/getting-started.md)
