[Debug Agent](https://www.debug-agent.com) skill let's helps you to debug an issue using AI.

You can trigger it just by `/debug` command inside your coding agent like Cursor, Codex, Claude Code, etc

## How to get started?

```shell
npx debug-agent@latest init
```

It's [open source](https://github.com/millionco/debug-agent) as well.
