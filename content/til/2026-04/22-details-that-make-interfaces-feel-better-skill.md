[Jakub Krehel](https://jakub.kr/) has released the design choices that he makes to build a good UI as a SKILL which we can use to make the AI generated UI better :) 

## How to get started?

```shell
npx skills add jakubkrehel/make-interfaces-feel-better
```

After it is installed you can invoke it with `/skill make-interfaces-feel-better` slash command in your AI coding agent like Claude Code.

👉 https://jakub.kr/writing/details-that-make-interfaces-feel-better

In his site, he has a interactive UI where you can playaround to see the different in UI because of those choices.
