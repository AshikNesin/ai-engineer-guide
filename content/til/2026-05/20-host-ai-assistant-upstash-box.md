[Upstash Box](https://upstash.com/docs/box/overall/quickstart) nows supports option to turn off auto-pass which means you can host AI assistants like OpenClaw or Hermes in it for fixed price like $8+/mo.

![image.png](https://cdn.qblog.nesin.io/f_auto,q_auto/qblog/AIEngineerGuide/2026-05/zw9t0xgnir7j9xecpvzy)

The main advantage is that it is completely isolated so you don't have to worry about security.

Once you've created your Upstash Box, you can login ito it via SSH and install the OpenClaw or Hermes in it and then you can use it.
