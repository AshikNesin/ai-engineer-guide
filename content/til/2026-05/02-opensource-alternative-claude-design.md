Anthropic has recently launched [Claude Design](https://www.anthropic.com/news/claude-design-anthropic-labs) which let's you create a good looking designs and using it you can quickly create prototypes.

But the problem is you can only use their Opus 4.7 model. And it is heavily vendor locked.

If you're looking for an alternative then [open-design](https://github.com/nexu-io/open-design) might be a good choice.

![image.png](https://cdn.qblog.nesin.io/f_auto,q_auto/qblog/AIEngineerGuide/2026-05/i29tx4m4cljzv2549j3a)

Under the hood it leverages existing skills [alchaincyf/huashu-design](https://github.com/alchaincyf/huashu-design), [OpenCoworkAI/open-codesign](https://github.com/OpenCoworkAI/open-codesign), etc
