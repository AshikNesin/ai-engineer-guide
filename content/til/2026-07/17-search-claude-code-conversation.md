I learned a slick keyboard shortcut for Claude Code to search a text in conversation.

Just press `Ctrl + O`

| Key | Action |
|-----|--------|
| `/` | Search for a term |
| `𝚗` | Step through the matches |
| `}` | Hop between your own prompts |
