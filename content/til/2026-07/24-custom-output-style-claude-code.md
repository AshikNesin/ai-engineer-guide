Claude Code lets you configure your own output style. You can prompt it with the style that you prefer.

We'll need to add it in `~/.claude/output-styles` directory.

And select that output style by running `/config` then selecting **Output style**

Here is how Lydia Hallie does it for to make it respond with ELI5 (explain like I'm 5) format.
