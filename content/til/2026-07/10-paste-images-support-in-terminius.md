If you prefer to use Claude Code or similar AI agents on the go using mobile apps like [Termius](https://termius.com/), the one problem that you would face is sending image as context.

You'll have to manually upload the file somehow using services like [glance](https://glance.sh/) or similar approach.

Termius for iPhone/iPad recently has first class support for image upload support.
