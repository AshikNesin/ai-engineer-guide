[Pi](https://pi.dev) has first class support for [Llama.cpp](https://github.com/ggml-org/llama.cpp) released in version 0.81.0+

{{< video "https://cdn.qblog.nesin.io/qblog/AIEngineerGuide/2026-07/nhrs3cakswy9rjq1ybob.mp4" >}}

## How to get started?
You can use the following slash command to do a one time setup.

```
/login llama.cpp 
```
