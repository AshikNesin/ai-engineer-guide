The Clineteam has launched [Kanban](https://cline.bot/kanban) - CLI-agnostic (Claude Code/Codex/Cline) multi-agent orchestration.

In simple terms you install their cli and you get a minimal Kanban board which uses the coding agents under the hood to do complete your task.

{{< video "https://cdn.qblog.nesin.io/qblog/AIEngineerGuide/2026-03/acmvqsxuglqddqzrazje.mp4" >}}


## How to get started?

```shell
npm i -g cline
```

![2026-03-29-at-22.41.302x.png](https://cdn.qblog.nesin.io/f_auto,q_auto/qblog/AIEngineerGuide/2026-03/zxzq4lxcjtkblxn4hscn)

Once you access the local server, you can configure the agent to use (in my case I'm using Claude Code)

![2026-03-29-at-22.44.302x.png](https://cdn.qblog.nesin.io/f_auto,q_auto/qblog/AIEngineerGuide/2026-03/plmv48inrfxkxgicq9sl)

And just give a task to complete. It'll complete it under the hood for you.

![2026-03-29-at-22.28.402x.png](https://cdn.qblog.nesin.io/f_auto,q_auto/qblog/AIEngineerGuide/2026-03/pr0ol3hupcaowkdea52v)

I really like their slick ui as well.

![2026-03-29-at-22.36.342x.png](https://cdn.qblog.nesin.io/f_auto,q_auto/qblog/AIEngineerGuide/2026-03/ikvyunt7pgllyexlehuc)

Like all the other works on Cline team this is also an [open source](https://github.com/cline/kanban) as well.
