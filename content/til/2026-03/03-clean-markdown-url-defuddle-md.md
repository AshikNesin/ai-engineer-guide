Defuddle the package using which you can get [clean content](https://aiengineerguide.com/til/extract-clean-content-with-defuddle/) from a web pages/blog now has a support for web app as well.

Just append `https://defuddle.md/` before your url.

{{< video "https://cdn.qblog.nesin.io/qblog/AIEngineerGuide/2026-03/naejslkcx3vgraf0qwnk.mp4" >}}


And you get it's content in **clean markdown** format.
