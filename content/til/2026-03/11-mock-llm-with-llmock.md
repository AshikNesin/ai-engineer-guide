[llmock](https://llmock.copilotkit.dev/) allows you to mock the LLM inference. You just need to point your base server to it and rest of the things will work as expected.

This is especially useful if you've some sort of testing where you do not need to hit the actual LLM API.

![2026-03-11-at-20.00.27.png](https://cdn.qblog.nesin.io/f_auto,q_auto/qblog/AIEngineerGuide/2026-03/m1ao4l2pupdfnnrktmnc)

It has support for all the inference format like OpenAI chat completion, responses message type, Anthropic, etc

![2026-03-11-at-20.05.51.png](https://cdn.qblog.nesin.io/f_auto,q_auto/qblog/AIEngineerGuide/2026-03/f9i0pi7yeawrono3bims)

And has wide range of features as well
