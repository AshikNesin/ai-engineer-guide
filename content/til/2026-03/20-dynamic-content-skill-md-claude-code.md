You can pretty easily add dynamic context just by adding **!`command`** in your SKILL.md

Claude Code will run those commands and replace it with actual value in the SKILL.md

![image.png](https://cdn.qblog.nesin.io/f_auto,q_auto/qblog/AIEngineerGuide/2026-03/gxtb1uzl6b9z2ykcimux)

Even though, this feature is really cool and address the day to day problem. 

It might open up vulnerability when you use some random SKILL.md from the internet.

Always read the SKILL before you plan on using it.
