[Pi.dev](https://pi.dev/) is one of the best coding agents which is really minimal but performs as good as/better than other coding agents like Claude Code, Codex, Droid, etc.

And I prefer using it over other AI agents whenever possible. It just gets things done and if you want something (your own custom workflow inside it), you can just ask it to build it and it'll build a extension to solve that problem.

Alejandro has a very detailed architecture walkthrough of Pi and it's underlaying packages like pi-core, etc

In case, if you wanted to understand how Pi works under the hood, check out the video 👇
