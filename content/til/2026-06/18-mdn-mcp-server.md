[MDN](https://developer.mozilla.org/en-US/) is now available as MCP server. 

![image.png](https://cdn.qblog.nesin.io/f_auto,q_auto/qblog/AIEngineerGuide/2026-06/ogejjzt0jytib4pm2hzc)

If you're a someone who builds frontend apps then it is huge time saver. 

## How to get started?
The server is at `https://mcp.mdn.mozilla.net/`

In Claude Code you can install it using following command

```shell
claude mcp add --transport http mdn https://mcp.mdn.mozilla.net/
```
