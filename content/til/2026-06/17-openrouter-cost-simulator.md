If you primarily use OpenRouter for inference and want to see how much the cost would be for your usage across different model then [Cost Simulator](https://openrouter.ai/labs/simulate-price) feature might be a good choice for you.

Just select the source model and then the model agaist which you want to compare the price agaist.

And it'll show the difference like this

![image.png](https://cdn.qblog.nesin.io/f_auto,q_auto/qblog/AIEngineerGuide/2026-06/cilcbplnxosub5jtycrs)

Note: You'll be able to use this feature only once you're logged in. And you've used any models in it recently.
